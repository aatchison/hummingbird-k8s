=== notes ===
- `hbird nodes` uses `cp_kubectl_raw` from the shared `cp_kubectl.rs`
  module (sibling of `commands/`). Same SSH wiring as cycle 1
  drain/uncordon (PR #325).
- No divergence from bash for the read-only kubectl pass-through —
  this is the cleanest demo of "no rewrite, just route the bytes
  through SSH and print them".
